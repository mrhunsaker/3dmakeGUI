# 3DMake GUI Wrapper

Welcome to the official documentation for **3DMake GUI Wrapper**, an
accessibility-focused NiceGUI frontend for
[tdeck/3dmake](https://github.com/tdeck/3dmake).

This site is intended for both users and contributors. It covers setup,
security guidance, containerized deployment, contribution workflow, code style,
and API-level reference documentation generated from NumPy-style docstrings.

## Highlights

- Accessibility-first design with keyboard and screen-reader support
- Integrated command streaming and line-numbered output dialogs
- SVG preview and interactive STL visualization
- Packaged desktop builds (PyInstaller)
- Containerized deployment options (Docker and Podman)
